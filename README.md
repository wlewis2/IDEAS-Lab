# IDEAS Lab

Landing page for the **IDEAS Lab** — Imaging, Devices, Engineering and Applications for Skin.

This is a static website (a single `index.html` file) hosted with **Cloudflare Pages**, which automatically rebuilds and republishes whenever changes are pushed to the `main` branch.

## How it works

- `index.html` — the entire site (HTML + CSS, no build step required)
- Pushing to `main` triggers Cloudflare Pages to deploy automatically (live in ~1 minute)

## Editing

Changes are made through Claude Code: describe the change in plain English and it will be committed and pushed.
